# save( boolean|null $force_update = false )

Saves this object's properties to the Edge server.

Saves this object's properties to the Edge server.

If $force_update is set to true, we assume that this is an update call. If it is false, we assume that it is an insert. If null is passed in, we attempt an update, and if it fails we attempt an insert. This is much less efficient, so declaring $force_update as a boolean will yield faster response times.
