# loadByAppId( string $appId, boolean $reset_developer = false )

Loads a DeveloperApp/CompanyApp, given its appId (which is a UUID).

Loads a DeveloperApp/CompanyApp, given its appId (which is a UUID).

Normally you'd find an app by listing its owner entity's apps and looking for the name you want. However, if you already know the app's unique id, you can load without knowing its owner.

If you pass true as the second parameter here, the DeveloperApp/CompanyApp object will be changed so that it pulls apps from this developer/company by default.
