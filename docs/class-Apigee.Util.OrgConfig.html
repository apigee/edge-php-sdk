Class OrgConfig

A class that represents infomration about an Edge organization.

Namespace: Apigee\Util
Located at Util/OrgConfig.php
Methods summary
public
# __construct( string $org_name, string $endpoint, string $user, string $pass, array $options = array() )

Create an instance of OrgConfig.

Create an instance of OrgConfig.

The $options argument is an array containing the fields 'logger', 'user_email', 'subscribers', and 'http_options'.

For example:

  $logger = new Apigee\Drupal\WatchdogLogger();
  $logger::setLogThreshold($log_threshold);
  $user_mail = "me@myCo.com";

  $options = array(
    'logger' => $logger,
    'user_mail' => $user_mail,
    'subscribers' => null,
    'http_options' => array(
      'connection_timeout' => 10,
      'timeout' => 50
    )
  );
